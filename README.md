# 🤖 Freelance Hunter Bot

Telegram бот для поиска вакансий на HH.ru с возможностью подписки на ежедневную рассылку.

## 📋 Функционал
- 🔍 Поиск вакансий по ключевым словам
- 📅 Ежедневная рассылка новых вакансий в 13:00 МСК
- ✅ Подписка на интересующие ключевые слова
- ❌ Отписка от рассылки
- 📊 Просмотр активных подписок
